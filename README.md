# Eternal Arena

A polished 2D fantasy fighting game featuring fluid combat, cinematic effects, and responsive controls.

## Overview

**Eternal Arena** is a 2D side-view fantasy fighting game inspired by classic arena fighters. Players choose from 6 unique characters (Knights, Assassins, Mages) and battle AI opponents in beautiful fantasy arenas.

### Features

- **6 Playable Characters**: 3 classes × 2 genders with unique movesets
- **5 Fantasy Arenas**: Ancient Temple, Lava Fortress, Frozen Kingdom, Forest Ruins, Celestial Arena
- **Deep Combat System**: 
  - Movement (walk, run, dash, jump, double jump, crouch)
  - Defense (block, parry, dodge roll)
  - 7 attack types per character
  - Combo mechanics with damage multipliers
  - Ultimate abilities with cinematic effects
- **AI Opponents**: Three difficulty levels with intelligent behavior
- **Progression System**: Unlock characters, weapons, armor, and cosmetics
- **Cinematic Presentation**: Dynamic camera, particle effects, screen shake, slow-motion

## Game Modes

- Single Player
- Practice Mode
- AI Battle
- Survival Mode (optional)
- Local Versus (optional)
